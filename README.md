# Sentiment Analysis: Bag of Words vs TF-IDF

## Project Overview
A complete sentiment analysis system comparing BoW and TF-IDF 
feature extraction techniques using NLTK Movie Reviews dataset.

## Dataset
NLTK Movie Reviews Corpus
- Total Reviews: 2000
- Positive Reviews: 1000
- Negative Reviews: 1000

## Installation
pip install -r requirements.txt

## Methodology
1. NLP Preprocessing Pipeline
2. Feature Extraction (BoW and TF-IDF)
3. Logistic Regression Model Training
4. Model Evaluation and Comparison

## Results
| Metric    | BoW    | TF-IDF |
|-----------|--------|--------|
| Accuracy  | 82.00% | 82.75% |
| Precision | 81.77% | 82.67% |
| Recall    | 82.59% | 83.08% |
| F1 Score  | 82.18% | 82.88% |
